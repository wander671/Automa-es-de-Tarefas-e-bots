# 🚀 Automação de Cadastro de Produtos

Projeto em Python que automatiza o cadastro de produtos em um sistema web utilizando **PyAutoGUI** e leitura de dados via **CSV**.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido para automatizar tarefas repetitivas de cadastro de produtos em um sistema online.

A automação realiza:

* Abertura do navegador
* Acesso ao sistema
* Login automático
* Leitura de uma base de dados (CSV)
* Preenchimento automático dos campos
* Cadastro de múltiplos produtos

---

## 🛠️ Tecnologias utilizadas

* Python
* PyAutoGUI
* Pandas

---

## ⚙️ Funcionalidades

✅ Automação de login no sistema
✅ Leitura de dados com Pandas
✅ Preenchimento automático de formulário
✅ Cadastro em lote de produtos
✅ Script para captura de coordenadas do mouse

## 📦 Instalação

Clone o repositório:

```
git clone https://github.com/SEU_USUARIO/automacao-cadastro-produtos.git
```

Instale as dependências:

```
pip install -r requirements.txt
```

---

## ▶️ Como executar

1. Execute o script para pegar as coordenadas do mouse:

```
python pegar_posicao_mouse.py
```

2. Ajuste as coordenadas no código principal (`main.py`)

3. Execute a automação:

```
python main.py
```

---

## ⚠️ Observações importantes

* As coordenadas do mouse podem variar de acordo com a resolução da tela
* O tempo de carregamento da página pode impactar a automação
* Não é recomendado expor senhas diretamente no código
